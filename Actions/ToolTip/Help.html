ToolTip.TrayBalloon

ToolTip.TrayBalloon

 

ToolTip.TrayBalloon ( 

(string) Titulo,

(string) Mensaje,

(number) Duración,

(string) IconoBS,

(number) Icono )

Description

Crea un tooltip tipo trayballoon en la bandeja del sistema

Parameters

Titulo

(string) El mensaje que se ubica en la parte superior del tooltip.

Mensaje

(string) Es el contenido que se transmite al usuario.

Duración

(number) Es el tiempo en milisegundos que dura el mensaje visible 1000 mil es igual a 1 segundo.

IconoBS

(string) Ruta del icono que sera visible en la bandeja del sistema.

Icono

(number) Es el icono del tooltip en el mensaje.

CONSTANT

VALUE

DESCRIPTION

Returns
