(table) Colors.InOut ( 

(table) _STARTCOLOR = {30, 64, 47},

(table) _ENDCOLOR = {230, 164, 147},

(number) _SPEED = 3 )

Description

This action create a fade effect between two colors (only for RGB table color)

Note: Arguments must be a type table, table, number, string & string (not required)

Parameters

_STARTCOLOR

(table) This action set the start color to move to _ENDCOLOR

_ENDCOLOR

(table) This action set the end color to move from _STARTCOLOR

_SPEED

(number) This action set the speed between each change of color

Returns

(table) This action return a table if success with each time the new Decimal and RGB channels color and false if an error occure.

-- **********************************************************
-- GLOBALS VARIABLES USED
-- CONSTANTS NAMES MUST BE USED AS THIS:
-- _tCOLORS (.Dec, .Red, .Green, .Blue) - Dynamic ( each time a color change _tCOLORS change also)
-- _tCOLORSFULL ( each row has the new color ) - Static (at the end of the function)
-- _SHAPEOBJECT ( optional: to set color to a shape object )
-- _LABELOBJECT ( optional: to set color to a text label )
-- **********************************************************
--   YOU MUST USE THIS CONSTANTS VARIABLES NAMES ONLY
-- **********************************************************


Example

-- Exemple 1 (using no optional arguments and using Page *On Timer* event )
-- Fade Color A to Color B with speed 3 and return each time the new color in table _tCOLORS:
-- Start the page timer ( time in page timer must be the same as in speed function argument 3)
Page.StartTimer (3, 10);

-- Fade InOut
Colors.InOut({255,128,0}, {128, 255, 40}, 3);

-- Stop the page Timer
Page.StopTimer(10);
------------------------------------------------------
* On Timer: event script for exemple *
------------------------------------------------------
-- Set a color to a shape each time color change
-- This script is 'On Timer' Event of a page or a dialog:
if ( _tCOLORS ~= nil ) then
  Shape.SetFillColor("myshape", _tCOLORS.Dec);
end
------------------------------------------------------

-- Exemple 2 ( using optional hidden arguments, shape and label):
_SHAPEOBJECT = "mySHAPE";
_LABELOBJECT = "myLABEL";
Colors.InOut({255,128,0}, {128, 255, 40}, 3, _SHAPEOBJECT , _LABELOBJECT );
-- so each time the color change:
then the shape color and the label change also,
with new values ( Shape receive a Decimal color
while label receive a string RGB color channel in
that format: {255,128,30}

*Note: You can use One or Two optional argument,
if you use the first (for shape) the second (for label)
can be skipped, if you use only the second (for label) then
the first (for shape) must be = nil as in Exemple 3)

-- Exemple 3 (using *note)
-- you must use this variables names only.
-- Set argument(s)
_SHAPEOBJECT = nil
_LABELOBJECT = "myLABEL";

-- run function
Colors.InOut({255,128,0}, {128, 255, 40}, 3, _SHAPEOBJECT , _LABELOBJECT );

-- Exemple 4 (using only shape as optional argument)
-- Set argument(s)
_SHAPEOBJECT = "mySHAPE";

-- run function:
Colors.InOut({255,128,0}, {128, 255, 40}, 3, _SHAPEOBJECT);

--****************************************************************************************--
--* If you dont use Object argument(s), 'On Timer' Event need to be used as in Exemple 1 *--
--****************************************************************************************--
