Colors.Convert


(table) Colors.Convert ( 

(string) _ARG1 = "DEC",

(string) _ARG2 = "RGB",

(table) _ARGS = {65430} )

Description

Old name: ColorConversion, This action Convert from a color format to another. eg: Dec 2 Hex

Parameters

_ARG1

(string) Input color format to convert to _ARG2

_ARG2

(string) Output color format to convert from _ARG1

_ARGS

(table) Input(s) argument(s) depend on input format ( eg: for Dec format only 1 arg. in table but for RGB format it use 3 args. )


Return

This action returns a (table) value.

(table) Colors.InOut ( 

(table) _STARTCOLOR = {30, 64, 47},

(table) _ENDCOLOR = {230, 164, 147},

(number) _SPEED = 3 )

Description

This action create a fade effect between two colors (only for RGB table color)

Parameters

_STARTCOLOR

(table) This action set the start color to move to _ENDCOLOR

_ENDCOLOR

(table) This action set the end color to move from _STARTCOLOR

_SPEED

(number) This action set the speed between each change of color


Return

This action returns a (table) value.
