Plugin Name:

Menu

Current Version:

3.3.0.0

Description:

The Menu Action Plugin makes creating and manipulating menus a cinch.
Forget those exhaustive tables of properties and subtables. With the Menu plugin you'll be on your way to simple menus in no time at all. no more referencing menu items by number. No more miles of code in your On Menu events. This plugin allows you to reference your menu items by name and to specify code based on those name so you never need to remember a menu id again.

Menu.SetItem


(number) Menu.SetItem ( 

(string) Menu Item = "File|Open",

(number) Icon ID = 0,

(boolean) Enabled = true,

(boolean) Checked = false,

(string) Code = "Dialog.Message(\"Test\", \"Well, it seems to be working.\r\nAre you impressed?\", MB_YESNO, MB_ICONQUESTION, MB_DEFBUTTON2);",

(boolean) AutoUpdate = true )

Description

This action is used to create and edit menu items.

Note: You can use this to create new menu items or change any of the properties of an existing item. To specify a separator, use "---" as the item (e.g. "File|---").

Parameters

Menu Item

(string) The path to the menu item.

Icon ID

(number) The icon id used for this menu item

Enabled

(boolean) Whether or not the item will be enabled.

Checked

(boolean) Whether or not the item will be checked.

Code

(string) The code that will be executed when the menu item is selected.

AutoUpdate

(boolean) Whether or not to auto-update after the action is complete.

Note: This can be left nil.

Returns

(number)
Returns 0 if the action was successful.
Returns -2 if the item was constructed incorectly.
Returns -3 if there is no active menu.
